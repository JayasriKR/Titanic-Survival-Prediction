# Titanic-Survival-Prediction
Machine Learning project to predict Titanic survival
# Titanic Survival Prediction
This project uses Machine Learning to predict whether a passenger survived the Titanic disaster.
# Project Overview
In this project, I performed data preprocessing, feature selection, and applied multiple machine learning models to predict survival outcomes.
# Dataset
- Dataset: Titanic Dataset
- Features used:
  - Pclass (Passenger Class)
  - Sex
  - Age
# Steps Performed

1. Data Cleaning
   - Filled missing values in Age column using mean
2. Data Preprocessing
   - Converted categorical data (Sex) into numerical values
3. Feature Selection
   - Selected important features: Pclass, Sex, Age
4. Train-Test Split
   - Split data into training and testing sets (80%-20%)
5. Model Training
   - Logistic Regression
   - Decision Tree
   - Random Forest
6. Model Evaluation
   - Accuracy Score
   - Confusion Matrix
   - Classification Report
# Models Used
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
